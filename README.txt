Инструкция для Windows

Скрипт с базой находится в папке MySQL_Script

Для запуска проекта необходимо собрать проект посредством Maven:
Я это делал через IDEA, в правой части экрана есть вертикальная панель, в ней выбираем Maven, в папке Lifecycle запускаем package.

После того, как проект собран, необходимо перейти в папку target и запустить файл parts-0.1.jar через консоль:
1. в папке target зажимаем Shift + ПКМ -> Открыть окно команд
2. прописываем: java -jar parts-0.1.jar
(отмечу, что путь к jdk должен быть прописан в переменных средах)

Когда проект загрузится, переходим в браузере по пути: http://localhost:8080


Примечание: при запуске возможна ошбика связанная с глобальной переменно mysql: caching_sha2_password
(подобная ошибка вылетала у меня на другом ноутбуке, как я понял это связано с версиями mysql)
Решение: в mysql прописать следующее: ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'root';
